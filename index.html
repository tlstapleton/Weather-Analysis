Summary: Latitude vs. X

In this analysis, several weather factors are measured in relationship to distance from the equator. This analysis is based on data pulled using the Open Weather Map API to aggregate data on weather variables from over 500 cities around the world.

After assembling the dataset, we used Matplotlib to plot the relationship between each measured weather variable for each city and the city's distance from the equator, also known as the city's latitude. The factors chosen for analysis were : Humidity, Cloudiness, Max Temperature, and Wind Speed. This site provides the source data and visualizations of all the data collected, as well as descriptions of each analysis.
